UrbanSimAI Chicago

A free smart-city demo that forecasts hourly average traffic speed by ZIP code for the next 24 hours, built using XGBoost + open Chicago datasets with a Streamlit dashboard and map.

What you can do in the app

  • Select a Chicago ZIP code and view trends
  • Generate a 24-hour forecast with chart + table
  • Use Backtest mode to compare predicted vs actual
  • Explore a map of latest observed hourly average speed by ZIP

Open data sources

  • City of Chicago Traffic Tracker (historical + latest snapshot)
  • CTA GTFS (transit stops)
  • Zillow ZHVI (home values by ZIP)
  • OpenStreetMap roads (OSMnx)
